# Información del Proyecto Base

## Lenguaje usado
**Java**

## Versión del lenguaje
**Java 21** (configurado en pom.xml)

## Framework usado
**Spring Boot 3.4.5**

## ORM instalado
**Spring Data MongoDB** (spring-boot-starter-data-mongodb)

## Motor de base de datos usado
**MongoDB Atlas** (Cloud)

## Comando usado para crear el proyecto
```bash
# Usando Spring Initializr o IDE
# Dependencias: Spring Web, Spring Data MongoDB, Validation
```

## Comando usado para instalar dependencias
```bash
mvn clean install
# o
./mvnw clean install
```

## Dependencias principales
- spring-boot-starter-web
- spring-boot-starter-data-mongodb  
- spring-boot-starter-validation
- spring-boot-starter-test
